IPS – Plataforma iCUR@

Sistema integral para la gestión de Seguridad Patrimonial, accesos, incidencias, activos y mantenimiento institucional.

¿Qué es iCUR@?

iCUR@ es una plataforma informática centralizada que permite a instituciones educativas gestionar de manera eficiente la seguridad patrimonial, control de accesos, incidencias, inventarios, mantenimiento y riesgos, asegurando trazabilidad y control en tiempo real.

Módulos Principales

  • Control de Accesos (visitas, proveedores, eventos)
  • Gestión de Incidencias
  • Gestión de Riesgos
  • Mantenimiento de Seguridad
  • Gestión de Extintores
  • Inventarios y Activos
